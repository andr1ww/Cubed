# Cubed
A Fortnite Gameserver for the version **18.40**, created by [andr1ww](https://github.com/andr1ww) & [mariki](https://github.com/Mar1ki). 

## How to Use
1. Download repository
2. Open Cubed.sln in Visual Studio 2022+ 
3. Set build configuration from Debug to Release
4. Click Ctrl + B to build solution
5. Have fun!

## Features
- Bugha LateGame
- Base Player AI's (UAthenaAIServicePlayerBots)
- Base Quest & XP Logic
- AFortGameSessionDedicatedAthena & Backfill
- Proper Harvesting
- Basic Creative Plots
- and more!
